# VEDR Guide

A modern, interactive benefits decision guide portal built with React and Vite.

## Features

- **Modern UI**: Clean and minimalist design powered by Tailwind CSS.
- **Interactive Elements**: Smooth animations leveraging Framer Motion.
- **Icons**: Crisp, scalable iconography from Lucide React.
- **Responsive Navigation**: Seamless routing handled by React Router DOM.

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/johnsoninsights/VEDR-Guide.git
   ```
2. Navigate to the project directory:
   ```bash
   cd vedr-guide
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development Server

Start the local development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173/`.

### Building for Production

To create a production build:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```
